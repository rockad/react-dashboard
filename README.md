# Simple dashboard APP

## Backend
To run the backend on http://localhost:3000
```bash
cd backend
npm install
npn start
```

## Client
To run the client on http://localhost:8080
```bash
cd client
npm install
npn start
```

Client could be build into a static file with
```bash
cd client
npm run build
```

and then served with
```bash
cd client
npx serve -s build
```
